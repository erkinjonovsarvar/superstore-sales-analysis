# 🛒 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes retail sales data to uncover key business insights, profitability drivers, and performance trends. The goal is to support data-driven decision-making by identifying factors that impact sales and profit.

---

## 🎯 Objectives

* Analyze the relationship between discount and profit
* Identify top and worst-performing products
* Evaluate regional and segment performance
* Discover sales trends over time

---

## 🧰 Tools & Technologies

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Jupyter Notebook

---

## 📊 Key Insights

### 🔻 Profit vs Discount

Higher discounts significantly reduce profitability. Many high-discount transactions result in losses.

### 🏆 Category Analysis

Technology is the most profitable category, while Furniture shows lower profit margins.

### 📦 Sub-Category Analysis

* Top performers: Phones, Copiers
* Worst performers: Tables, Bookcases

### 🌍 Region Analysis

The West region generates the highest profit and sales performance.

### 👥 Segment Analysis

The Consumer segment contributes the most to overall revenue.

### 📈 Time Series Analysis

Sales show seasonal patterns, with fluctuations across months.
